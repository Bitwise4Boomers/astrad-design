ERP combines per-element power coherently across active elements (10·log10(N·P) referenced to 1W); it excludes individual element antenna gain, which is not published for ASTRAD. Range resolution = c·τ/2 (uncoded); ASTRAD's actual coded pulses (e.g. 8 µs complementary code, 0.25 µs baud) achieve finer resolution than the uncoded value shown. Unambiguous range = c / (2·PRF).
